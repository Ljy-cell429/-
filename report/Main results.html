RABiTPy 细菌运动分析复现报告

基于 Flavobacterium johnsoniae 相差显微视频的轨迹与统计特性分析

1. 实验环境与安装配置

本项目基于 RABiTPy v1.1.0，要求 Python 版本为 3.10.11。

# 安装 RABiTPy
pip install RABiTPy

# 安装指定版本的 Omnipose（关键依赖）
pip install git+https://github.com/kevinjohncutler/omnipose.git@63045b1af0d5217

2. 视频捕获与预处理（Capture）

加载示例视频 sample.avi，提取帧并进行尺度标定。

from RABiTPy import Capture

capture = Capture()
capture.load_video('sample.avi')
frames = capture.process_video_into_frames(
    pixel_scale_factor=0.166666,
    scale_units='um'
)

3. 细菌识别与分割（Identify）

采用 AI 驱动的 Omnipose 模型进行细菌掩膜分割，模型选用 bact_phase_omni（适用于相差成像）。

from RABiTPy import Identify

identify = Identify(capture_frame_object=capture)
identify.initialize_omnipose_model(
    model_name='bact_phase_omni',
    use_gpu=True
)
identify.apply_omnipose_masking(batch_size=100)

4. 轨迹追踪（Tracker）

基于质心坐标进行跨帧连接，并设定最小帧数和位移阈值以过滤噪声轨迹。

from RABiTPy import Tracker

tracker = Tracker(identify_object=identify)
tracker.link_particles(
    max_distance=50,
    max_memory=50
)
tracker.filter_particles(
    min_frames=500,
    min_displacement=10
)

5. 运动统计分析（Stats）

计算细菌瞬时速度，并拟合正态分布以评估其运动模式。

from RABiTPy import Stats

stats = Stats(tracker_object=tracker)
stats.calculate_speed_and_plot_mean(
    distribution_type='norm',
    ci_range=(5, 95)
)

6. 主要结论

  • 成功使用 Omnipose 模型在相差显微镜下实现了对 Flavobacterium johnsoniae 的自动识别与分割。
  • 通过轨迹连接算法，获得了多条长时程（≥500 帧）的细菌运动轨迹。
  • 速度统计分析表明，该菌株在观测时间内呈现持续滑行运动特征，平均速度稳定于特定区间内（具体数值需根据复现结果填写）。
  • 本复现验证了 RABiTPy 在细菌运动量化分析中的有效性，尤其适用于高通量视频数据处理。
